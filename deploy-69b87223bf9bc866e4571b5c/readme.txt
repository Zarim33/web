AURA CLIENT - INSTALLATION

OPTION 1: AUTOMATIC INSTALLER (Recommended)
1. Double-click "AuraClient-Fabric-1.21.jar".
2. Follow the on-screen instructions.
3. Launch Minecraft with the new profile.

OPTION 2: FABRIC MOD (Stealth Mode)
1. Install Fabric for Minecraft 1.21.
2. Place "AuraClient-Fabric-1.21.jar" into your %appdata%/.minecraft/mods folder.
3. Launch Minecraft with Fabric.

Compatible with Minecraft 1.21 - 1.21.11
Requires Fabric Loader.
